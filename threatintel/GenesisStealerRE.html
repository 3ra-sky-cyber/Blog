Genesis Stealer

Website Analysis

A website currently being shared through Discord claims to offer a downloadable beta version of a game. However, further analysis reveals that the site contains heavily obfuscated JavaScript, a common technique used to hide malicious code. Instead of providing a legitimate game, the download installs malware known as “Geniesis Stealer,” an infostealer designed to extract sensitive data from the victim’s computer. This includes stored passwords, browser data, and other personal information. The code contains a lot of comments with suggest the use of AI.

Analysis on VirusTotal

Downloading the file

The download of the file is done through dropbox.

First after downloading the exe from the webpage we will need to extract is content with uniextract.

The malware is electron packed so we need to install electron and unpack it.

PS C:\Blitzvoid\BlitzVoidLauncher Setup 2.1.1\Additional files\$PLUGINSDIR\app-64\resources> asar extract .\app.asar .\app

Now to start the analysis of the extracted files we will open VS Code. VS Code is usefull when dealing with obfuscated JavaScript, as we can set breackpoints and use the debugger to see whats going on.

We can go https://deobfuscate.io/ and deobfuscate the JavaScript.

Now we have to see where we can find interesting values and where they are referenced.

There is another big obfusctated string, so we need to type ctl+f to see where there is a function with this references.

After coping the new more readable script we can add breakpoints on the left if we see something interesting.

After that, we can run the JavaScript debugger and see what variables we have that may be useful.
